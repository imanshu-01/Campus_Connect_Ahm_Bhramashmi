## 📋 Overview

**PlaceBridge** is a modern, intelligent campus placement management system designed to streamline the recruitment process for students, placement cells, and companies. The platform leverages smart eligibility checking, real-time tracking, and a readiness scoring system to help students navigate campus placements with confidence.

### ✨ Key Features

- **🎯 Smart Eligibility Checker** — Automatically matches students with companies based on CGPA, backlogs, skills, and branch criteria
- **📊 Placement Readiness Score** — A credit-score style metric (0-100) that measures placement potential in real-time
- **📈 Unified Dashboard** — Track applications, interview rounds, and results from a single interface
- **🔔 Real-Time Notifications** — Instant alerts for new drives, shortlists, and offer letters
- **🤖 AI-Powered Assistant** — Interactive chatbot to answer placement-related queries 24/7
- **💼 Company Management** — Browse and apply to top recruiters with detailed eligibility criteria
- **📱 Fully Responsive** — Seamless experience across desktop, tablet, and mobile devices
- **🔐 Secure Authentication** — User session management with localStorage

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Local development server (optional but recommended)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/placebridge.git
   cd placebridge
   git clone https://github.com/yourusername/placebridge.git
   cd placebridge
   ```

2. Set up the project structure

```bash
placebridge/
├── index.html
├── studentlogin.html
├── dashboard.html
├── apply.html
├── company_detail.html
├── eligibilty.html
├── compny.html
├── notitication_page.html
├── img/
│   └── bg.jpg
└── README.md
```

Add background image

Run the application

```bash

```

```bash

```
